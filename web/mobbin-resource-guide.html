Theme Implementation

Mobbin uses an HSL-based token system: colors are stored as raw HSL values in CSS variables (e.g., --neutral-30), then composed with hsl(var(--neutral-30)) or hsl(var(--neutral-30)/50%) for opacity. The theme is toggled via a class on <html> with a localStorage-backed script in <head> to prevent FOUC.

Tech Stack

Mobbin is built with Next.js (App Router), React, Tailwind CSS with custom configuration (custom breakpoints, extended spacing), and uses Sentry for monitoring. Recreation Tip

The seamless loop trick: render the full list of items inside a flex shrink-0 container, then duplicate the entire container as a sibling. Both animate together via translateX(-100%). The overflow: hidden on the parent masks the duplication. The pr-24/pr-40 (padding-right) on the inner item group ensures proper spacing where the sets meet during the loop transition. Use will-change: transform for GPU-accelerated scrolling.
